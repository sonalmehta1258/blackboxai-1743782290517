# Rail Madad Admin Dashboard

## Project Overview
The **Rail Madad Admin Dashboard** is a web-based application designed for railway administration to manage and analyze complaints, medical cases, emergencies, and user activities efficiently. Utilizing modern web technologies, this dashboard provides a comprehensive visual representation of various metrics related to service performance and user reports, enabling a data-driven approach for decision-making.

## Installation
To set up the project locally, follow these steps:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/rail-madad-dashboard.git
   cd rail-madad-dashboard
   ```

2. **Open the HTML file**
   Open the `dashboard.html` file in your web browser to view the dashboard. There are no specific server setup requirements since the project can be run locally by opening the HTML file directly.

## Usage
The dashboard provides several sections that can be navigated using the sidebar:
- **Main Dashboard**: View overall statistics including total complaints, pending resolutions, medical cases, and emergency alerts.
- **Complaints Management**: Analyze complaints by type and view details of recent complaints.
- **Analytics Section**: Insights into complaints resolution times and detailed visualizations through charts.

The sidebar can be toggled to expand or collapse, providing a cleaner interface for navigation.

## Features
- **Responsive Design**: Adapts to various screen sizes for improved usability on mobile devices and desktops.
- **Dynamic Charts**: Interactive visualizations using Chart.js to represent complaints by type and resolution duration.
- **Real-Time Stats**: Displays up-to-date statistics for efficient monitoring of key performance indicators.
- **Sidebar Navigation**: Easily navigable sidebar with links for major sections of the dashboard.

## Dependencies
The project uses the following libraries for styling and interactivity:
- **Tailwind CSS**: For utility-first CSS styling.
- **Chart.js**: For data visualization.
- **Font Awesome**: For scalable vector icons.

No external package manager dependencies are required as this project primarily relies on CDN links for libraries.

## Project Structure
The project's file structure is straightforward, containing:
```
/rail-madad-dashboard
│
├── dashboard.html
```

- **dashboard.html**: The main HTML file that contains the UI layout, styles, and JavaScript for interactivity and data visualization.
